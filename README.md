## рҹҡҖ Getting Started

### 1. Clone the Repository

```bash

```

### 2. Load the Extension

1. Open **Google Chrome**
2. Visit:

```

```

3. Enable **Developer Mode**
4. Click **Load unpacked**
5. Select the project folder
6. Start using **Smart Leads Tracker**
